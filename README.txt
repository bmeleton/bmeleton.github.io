# GitHub Pages Website Starter

Upload the contents of this folder to your `username.github.io` repository.

## Included pages
- Home
- Personal
  - Curriculum Vitae / Resume
  - Artwork
- Software
  - TeBATSim
  - TeBATResponse
  - TeBATAnalysis
  - TeBATDrift
- Tutorials & Courses
  - Geant4
  - Magboltz & Garfield++
  - ROOT
  - OpenMC

## Customize
- Replace `Ben` in the HTML files with your preferred full name.
- Add PDFs to a `files/` folder and update the download links.
- Add more images to `assets/`.
- Edit `styles.css` to change colors, spacing, or typography.
